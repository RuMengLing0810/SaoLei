## 📘 项目简介  
这是一个使用 **C 语言** 编写的 **命令行版扫雷游戏**。  
玩家在 9×9 的棋盘上排查 10 个隐藏的地雷。  
项目逻辑清晰，结构分明，非常适合 C 语言初学者练习数组与函数模块化编程。

---

## 🧩 游戏规则  
- 游戏开始后，自动生成一个 9×9 的雷区，随机布置 10 个雷。  
- 玩家输入坐标（例如 `3 5`）进行排查。  
- 若踩中地雷，游戏结束 💥；  
- 若成功排除所有雷，游戏胜利 🎉。  
- 输入非法坐标会提示重新输入。
