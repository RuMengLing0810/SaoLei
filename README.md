🧠 项目简介
这是一个使用 **C语言** 编写的 **命令行版扫雷游戏**。  
玩家在 9×9 的棋盘上排查 10 个隐藏的地雷 💥。  
游戏逻辑清晰，结构分明，适合 C 语言初学者练习数组与函数模块化编程。

🎮 游戏规则
游戏开始后，会生成一个 9×9 的雷区，随机布置 10 个雷。
玩家每次输入坐标（例如：`3 5`）进行排查：
如果踩中地雷，游戏失败 💀；
如果安全，则显示该格周围的雷数；
当所有非雷格被排查完毕，即胜利 🎉。
输入错误或重复坐标会有提示。
